# Readlog

I want to create a personal reading management app.

## Reason

There are three major reasons for creating this app.

1. I want to create somthing to develop healthy reading habit.
2. A good portfolio piece.
3. Mastering `React`, `Tailwind CSS`, `Databases i.e SQL`, and `ExprssJS` while developing this. 

## Final Look

I'm a bit too ambitious regarding this app. I want it to:
- Be Responsive and smooth
- Be User friendly with good visuals and accessablity.
- Design should feel attractive and personlized.
- Readlog: The nostalgic page, the motivation behind the name.
- Interact with OpenLibrary API for Meta data and covers.
- Interact with GutenBerg (Using some wrappers) for free books.
- Reading view
- Download free books.
- Authentication
- ...
